Phpcraft\Plugin::on (   $event_name,
  $function,
  $priority = \Phpcraft\Event::PRIORITY_NORMAL

Defines a function to be called to handle the given event. Only one function can be defined per event per plugin, so subsequent calls with the same event name will overwrite the previously defined function.

Parameters
string$event_nameThe name of the event to be handled. Use an asterisk (*) to catch all events, and a period (.) to catch all uncaught events.
function$function
integer$priorityThe priority of the event handler. The higher the priority, the earlier it will be executed. Use a high value if you plan to cancel the event.
