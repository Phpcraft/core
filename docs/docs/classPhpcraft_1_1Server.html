◆ $list_ping_function

callable Phpcraft\Server::$list_ping_function = null

The function called when to get the server's response to a list ping request with the ClientConnection as argument. See Phpcraft::getServerStatus for an example of all the data a server may respond with (excluding "ping"). Additionally, if you set "no_ping", the client will show "(no connection)" where usually the ping in ms would be.
