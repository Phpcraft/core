Phpcraft\Plugin\Plugin::registerCommand (   $names,
callable  $function 
)
protected

Registers a command.

Parameters
$namesstring|string[] One or more names without the / prefix. So, if you want a "/gamemode" comand, you provide "gamemode", and if you want a "//wand" command, you provide "/wand".
callable$functionThe function called when the command is executed with the first argument being a CommandSender. Further arguments determine the command's arguments, e.g. ->registerCommand("gamemode", function(CommandSender &$sender, int $gamemode){...}) would result in the command /gamemode <gamemode> where the gamemode argument only allows integers.
