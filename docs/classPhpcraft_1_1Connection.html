Detailed Description

A wrapper to read and write from streams. The Connection object can also be utilized without a stream: 

$con = new Connection($protocol_version);
$packet = new ChatMessagePacket(["text" => "Hello, world!"]);
$packet->send($con);
echo hex2bin($con->write_buffer)."\n";
