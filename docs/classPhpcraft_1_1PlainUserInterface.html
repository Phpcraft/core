Detailed Description

◆ getMissingDependencies()

static Phpcraft\PlainUserInterface::getMissingDependencies ( )
static

Returns an array of dependencies required for spinning up a UI which are missing on the system. To spin up a plain UI, users need tput, which should be available because WINNT is no longer supported. Regardless, make sure the return of this function is an empty array before you initalize a PlainUserInterface.

Returns
array
